# 📧 Automated Resume Email Sender

A Python-based automation tool that sends job application emails with a resume attachment to multiple HR email addresses. The script reads recipient emails from a text file, creates professional email messages, attaches a PDF resume, and sends them securely using Gmail SMTP.

## 🚀 Features

* 📨 Send emails to multiple HR contacts automatically
* 📄 Attach resume (PDF) to every email
* 🔒 Secure Gmail SMTP authentication
* 📋 Read recipient emails from a simple text file
* ⚡ Saves time when applying to multiple companies

## 📁 Project Structure

├── send_mail.py          # Main Python script
├── emails.txt            # List of HR email addresses
└── resume_name.pdf # Resume attachment

## 🛠️ Technologies Used

* Python
* smtplib
* EmailMessage
* Gmail SMTP

## 📌 Summary

This project automates the job application process by sending personalized emails with resume attachments to multiple HR contacts. It demonstrates practical use of Python for email automation, file handling, and SMTP communication.
